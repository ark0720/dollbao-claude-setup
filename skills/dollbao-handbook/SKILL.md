## 觸發時機

任何涉及公司規章 / 內部 SOP 的問題：
- **人事：** 請假、加班、考核（DOC 貢獻面談）、晉升（逗寶之星）、福利（員工旅遊補助）、薪資、年資、班別調整
- **財務：** 費用報銷、資金分配申請單、採購單、Dolly 助手、零用金、收入檢核表
- **雲端硬碟：** 使用規則、檔案命名、共用圖檔位置、Drive 結構、原廠圖檔位置
- **品牌：** 企業色 PANTONE、品牌定位、品牌關鍵字、Slogan、目標客群、VFP
- **組織：** 組織圖、職務 / 職責 / 職位 / 職階定義、處別、CSW、會議制度（董事會 / 主管議會 / 諮詢議會）、書面溝通主旨格式、報告年週月、規章撰寫
- **其他全公司 SOP：** 用印、合約、書籍借閱、會議中心預約、信件包裹收發
